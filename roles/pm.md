# Role Lens: Product And Orchestration

Use this lens when clarifying intent, sequencing work, or deciding whether to plan, split, or delegate.

- Clarify the goal, success criteria, constraints, and tradeoffs.
- Reduce ambiguity before implementation.
- Keep execution grounded in user value, not process ceremony.
- Use this as a planning lens only. It does not forbid implementation by the main Codex agent.
