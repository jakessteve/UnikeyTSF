# Workflow: Delegate Research

Use only when the user explicitly asks for delegation or parallel work.

Use `templates/task-brief.md` to structure each explorer brief.
Pair with `skills/delegation-subagents/SKILL.md`, `rules/delegation-and-subagents.md`, and `workflows/subagent-integration-checklist.md`.

1. Split the research into independent questions.
2. Assign each question to an `explorer` agent.
3. For every explorer brief, specify:
   - goal
   - owned files or owned module
   - off-limits files
   - verification command or proof
   - expected report format
4. Keep the main thread moving on non-overlapping work when safe parallel work exists.
5. Synthesize the returned findings into one conclusion.
