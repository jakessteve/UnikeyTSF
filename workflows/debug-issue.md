# Workflow: Debug Issue

Pair with `skills/debugging-forensics/SKILL.md`, `rules/evidence-and-scope.md`, and `rules/delivery-and-verification.md`.

1. Reconstruct the failing path from logs, tests, or reproduction steps.
2. List the leading hypotheses.
3. Run the cheapest evidence-producing checks first.
4. Narrow to a root cause before editing.
5. Verify the fix against the original failure and one nearby edge case.
